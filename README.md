# 🛒 Shree Krishna Enterprises - E-Commerce Web App

A full-stack e-commerce web application built using **React, Node.js, Express, and MongoDB** for selling two-wheeler spare parts.

---

## 🚀 Features

- 🏍️ Shop by Category (Petrol Pipes, Rubber Bushes, Brake/Clutch Parts, etc.)
- 🔍 Dynamic Product Listing
- 📦 Product Detail & Order Page
- 🛒 Add to Cart Functionality
- 📱 WhatsApp Order Integration (COD)
- 🔐 User Authentication (Login / Register)
- 📊 Admin Dashboard (Orders & Status Management - In Progress)
- 📜 Order History Tracking (Planned)

---

## 🧱 Tech Stack

### Frontend
- React.js
- React Router DOM
- CSS

### Backend
- Node.js
- Express.js
- MongoDB (Mongoose)

### Other Tools
- WhatsApp API (via URL)
- Git & GitHub
